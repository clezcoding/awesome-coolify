Manual

Paste MCP JSON

Drop this into your client config. Replace URL and token with real values from Coolify → Keys & Tokens.

{
  "mcpServers": {
    "awesome-coolify-mcp": {
      "command": "npx",
      "args": ["-y", "awesome-coolify-mcp"],
      "env": {
        "COOLIFY_URL": "https://coolify.example.com",
        "COOLIFY_TOKEN": "YOUR_COOLIFY_API_TOKEN"
      }
    }
  }
}
