# Sketch Manifest

## Design Direction

Friendly, community-oriented open-source branding that feels approachable and welcoming — clearly associated with Coolify through cobalt-violet (`#6b16ed`) and dark surfaces (`#101010`), with optional buttercup accent (`#fcd34d`). Independent identity as an MCP developer tool, not a copy of Coolify marketing assets.

## Reference Points

- [Coolify](https://coolify.io/) — color palette and self-hosted PaaS context
- MCP protocol — agent ↔ server bridge metaphor
- OSS community tools — warm, inviting, not corporate
