## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| agent → manifest tool | Untrusted action args; Zod .strict() validates |
| manifest tool → Coolify API | sync/diff cross the network boundary; creds from InstanceManager |
| Coolify API 404 → agent | Stale UUID hint injection path; must not leak unrelated manifest entries |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-17-02-01 | Tampering | manifest tool input args | high | mitigate | z.discriminatedUnion + .strict() on every action schema; superRefine on clear/sync prune confirm gates |
| T-17-02-02 | Information Disclosure | sync/diff response leaking tokens | medium | mitigate | sync/diff return only UUIDs/names/domains from API — never credentials; manifest has no token fields (17-01) |
| T-17-02-03 | Denial of Service | auto-retry on 404 causing rate limits | high | mitigate | D-15: no auto-sync, no auto-retry — recovery hint only; single 404 → single response |
| T-17-02-04 | Tampering | sync prune without confirm | high | mitigate | Prune requires confirm:true AND prune:true (D-13); without both, orphans retained and listed |
| T-17-02-05 | Information Disclosure | 404 hint leaking non-manifest UUIDs | low | mitigate | Hint only triggers when UUID exists in manifest; hint text is generic, no other manifest entries exposed |
| T-17-02-06 | Spoofing | sync targeting wrong instance | medium | mitigate | InstanceManager.resolveCredentials resolves per-request; explicit `instance` param wins (D-11 from Phase 15) |
