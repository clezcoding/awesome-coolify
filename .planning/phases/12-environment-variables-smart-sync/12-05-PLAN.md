## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Local filesystem → handler (env_file) | Handler reads .env from disk; path traversal risk if uuid-derived |
| MCP caller → handler (env_content) | Inline .env text crosses as untrusted string |
| Handler → Coolify API | Apply phase issues POST/PATCH/DELETE via client.ts |
| Coolify API → MCP response | Disposition env values ALWAYS masked (sync is control-plane) |
