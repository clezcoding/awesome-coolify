## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| agent → instance tool args | Untrusted add/update/delete input crosses here; Zod validates name regex, url, token shape |
| instance tool → process.env | import-env reads COOLIFY_URL+COOLIFY_TOKEN from env; never writes env |
| server boot → filesystem/registry | Soft-start must not crash; registry absence is normal |
