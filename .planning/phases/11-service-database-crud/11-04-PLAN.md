## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Agent -> MCP update/delete handler | Untrusted input; Zod .strict() + superRefine is the first gate (SAF-03) |
| MCP handler -> Coolify PATCH /services/{uuid} | Existing bearer auth + retry + error mapping |
| MCP handler -> Coolify DELETE /services/{uuid} | Destructive action gated by confirm (SAF-01) + safe defaults (SAF-02) |
| MCP handler -> local filesystem (compose_file on update) | Path traversal risk mitigated by fs.readFile (no shell), size cap 1 MiB |
| MCP handler -> Coolify GET /resources (for delete_preview) | Read-only; no destructive side-effect |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|

| T-11-04-02 | Tampering | delete with unsafe defaults | high | mitigate | Zod defaults delete_volumes/delete_configurations/docker_cleanup/delete_connected_networks to false (SAF-02, D-16). Handler passes parsed values to deleteService query params. Coolify API defaults these to true — MCP overrides to false at the Zod layer. |
| T-11-04-03 | Tampering | update payload validation | high | mitigate | Zod .strict() rejects unknown fields; curated allowlist (D-20) prevents mass-assignment of unintended fields. |
| T-11-04-04 | Spoofing | identity resolution by name | medium | mitigate | resolveServiceMutationUuid multi-match -> COOLIFY_AMBIGUOUS_MATCH, no mutation (D-18). Existing Phase 5 pattern. |
| T-11-04-05 | Tampering | compose_file path traversal on update | high | mitigate | Handler reads compose_file via fs.promises.readFile (no shell, no glob, no execution). 1 MiB size cap. Reject empty result. validateCompose before encoding. |
| T-11-04-06 | Information Disclosure | 409 conflicts passthrough on update | low | accept | Conflicts contain domain/resource names only (verified OpenAPI 409 schema); no secrets. |
| T-11-04-07 | Denial of Service | accidental delete via ambiguous name | medium | mitigate | COOLIFY_AMBIGUOUS_MATCH refuses mutation on multi-match (D-18) — agent must narrow to a single UUID. |
