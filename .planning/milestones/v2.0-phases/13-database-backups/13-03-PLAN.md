## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Agent → backup:update payload | User-supplied frequency, S3 fields cross into MCP; Zod validates before HTTP (D-18); frequency restricted to presets (Pitfall 1) |
| Agent → backup:delete | Destructive action; confirm gate enforced before HTTP (D-08, D-09) |
| Agent → backup:now | On-demand trigger; no confirm gate (D-10); reuses PATCH endpoint (D-12) |
| Handler → Coolify API | Outbound HTTP with bearer token; errors mapped via mapApiError |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-13-03-01 | Tampering | frequency cron injection on update | high | mitigate | backupFrequencyUpdateSchema is z.enum(presets) — cron strings fail Zod parse with COOLIFY_VALIDATION_ERROR before HTTP (Pitfall 1). |
| T-13-03-02 | Tampering | backup:delete without confirm | high | mitigate | validateBackupDeleteConfirm throws COOLIFY_CONFIRM_REQUIRED without confirm (D-08). |
| T-13-03-03 | Tampering | backup:delete with delete_s3: true without confirm | high | mitigate | validateBackupDeleteConfirm enforces D-09 — purging S3 artifacts is destructive, still requires confirm: true. |
| T-13-03-04 | Information Disclosure | S3 credentials in backup:update response | high | mitigate | maskBackupConfig explicitly masks S3 credential fields as *** unless reveal: true (D-15). |
| T-13-03-05 | Repudiation | backup:now trigger without audit trail | low | accept | Coolify records execution history (BAK-06 surfaces it); no MCP-side audit needed. |
| T-13-03-06 | Tampering | backup:now called on wrong schedule | medium | mitigate | Both parent DB identity AND scheduled_backup_uuid required (D-13); resolveDatabaseUuid throws COOLIFY_AMBIGUOUS_MATCH on multi-match (D-03). |
